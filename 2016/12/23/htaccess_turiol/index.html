Introduction

.htaccess是一个纯文本文件，存放着Apache服务器配置的一些指令，类似于Apache站点的配置文件.当.htaccess文件放置在一个需要通过Apache载入的项目目录下，.htaccess将会被Apache Web Server监测到并且执行。我们可以通过修改.htaccess文件来进行Apache Web Server的相关配置。它支持重定向、URL重写以及访问验证登功能。

Order命令

Order命令在apache配置中很常见，它决定了Apache处理访问规则的顺序。

  • Order Allow,Deny，Apache首先找到并应用Allow命令，然后应用Deny命令，以阻止所有访问。
  • Order Deny,Allow，Apache首先找到并应用Deny命令，然后应用Allow命令，以允许所有访问。
